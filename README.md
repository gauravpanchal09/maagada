# Hospital

## Required
- PHP `7.4+`
- composer
- mysql
- apache
- phpmyadmin

## Steps For Install
- Clone the repo.
- `cd hospital`
- Install required dependency: `composer install`
- Run migration and seeder: `php artisan migrate --seed`
- Run the development inverment: `php artisan serve`
